Entregable solicitado:
- Archivo de texto con enlace a repositorio y contenidos listos para subir.
- Código fuente Java (en src/) implementando clases y operaciones sobre lista enlazada simple para alumnos.

Sugerencia rápida para crear el repo y subir:
1. Crear repo en GitHub (nombre: linkedlist-academico).
2. En su máquina:
   git init
   git add .
   git commit -m "Initial commit - linked list implementation for course students"
   git branch -M main
   git remote add origin https://github.com/<su_usuario>/linkedlist-academico.git
   git push -u origin main

El repositorio debe contener:
- src/Student.java
- src/Course.java
- src/CourseManager.java
- src/Main.java
- README.txt (explicaciones)

Justificación de la inserción:
Se eligió mantener la lista ordenada por lastName (alfabéticamente) y usar idNumber como desempate. Ventajas: la nómina queda legible en orden alfabético y facilita listados y comprobaciones humanas. La inserción ordenada requiere recorrer la lista para encontrar el punto correcto y actualizar referencias 'next', preservando la integridad.

Validaciones implementadas:
- credits > 0 al crear/actualizar curso.
- idNumber único en la lista de cada curso al inscribir.
- Manejo de lista vacía, borrado de head / nodo intermedio / último, búsqueda sin resultados.

Uso con SublimeText:
Abra los archivos en SublimeText y siga los pasos de compilación indicados en README.txt.
